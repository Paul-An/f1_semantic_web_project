# F1 Semantic Web Project

A Semantic Web / Web Data project focused on the Formula 1 domain.

## Objective
Build an end-to-end pipeline for data collection, information extraction, knowledge graph construction, reasoning, and question answering over Formula 1 data.

## Current Structure
- `data/raw/`: raw collected data
- `notebooks/`: exploration and testing notebooks
- `src/`: project source code
- `outputs/`: generated outputs
